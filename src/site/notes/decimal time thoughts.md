# Divide the Day into 1000
```math
beat = 24 * 60 minutes / 1000
tenbeat = beat * 10
hunbeat = beat * 100

pomo = beat + tenbeat
1000/3 minutes / pomo 
```

## Observations
- beats are ~2min
- tenbeats are ~15min
- hunbeats are ~2h
- Pomodoro
    - Work for a tenbeat
    - rest for a beat
    - about 20 pomodoros in an 8h workday
