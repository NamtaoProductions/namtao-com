## Autonomous Car Levels

0. Driver
    - Regular manual car
1. Feet Off
    - Adaptive cruise control
2. Hands Off
    - Autopilot, lane following
    - (We are here)
3. Eyes Off
    - danger avoidance
4. Mind Off
    - Route following
5. Passenger
    - Robotaxi (Waymo etc)

## AI Levels

0. AUTHOR
    - _no AI use_
    - vscode/vim/etc
1. CREATOR
    - _I write the code_
    - copilot questions
2. CURATOR
    - _I read the code_
    - copilot autocomplete
3. VIBER
    - _I don't read the code_
    - Claude Code
